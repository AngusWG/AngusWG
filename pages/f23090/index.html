###已使用解决方案

  • #####出错后需要rollback，为了后续程序能运行，给每个涉及sql语句的函数用了装饰器。
def auto_rollback(func):    
    def wrapper(*args, **kwargs):    
        try:    
            return func(*args, **kwargs)    
        except Exception as err:    
            db.session.rollback()    
            log.error(err)    
            raise err    
    
    return wrapper
